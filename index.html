Face_Mask_Project

I created a face mask detection project using Python, Keras, OpenCV and MobileNet . The architecture I used here to train data is Convolutional Neural Network architecture (CNN). But instead of using traditional CNN, I used MobileNet architecture (as it takes very less computation power to run or apply transfer learning to). The image dataset to train the model is taken from Kaggle and some open-source image libraries and google images.
