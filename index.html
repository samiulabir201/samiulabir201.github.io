About Me

Data Engineer

Hello! I'm Samiul Islam, a seasoned Data Engineer skilled in end-to-end data engineering pipelines.
My journey begins with data extraction and ingestion using tools like Apache Kafka and Apache Nifi. I design robust storage solutions with databases like PostgreSQL, MySQL, MongoDB, and Cassandra. Transforming raw data into insights is my forte, utilizing Apache Spark and Apache Flink for large-scale processing. For data warehousing, I'm experienced in Amazon Redshift, Google BigQuery, and Snowflake. I excel in crafting data workflows with Apache Airflow and deploying with Docker and Kubernetes.
In real-time data streaming, I leverage Apache Kafka and Apache Flink, implementing data quality checks with tools like Apache Atlas and Collibra. I specialize in data visualization using Tableau, Power BI, and Apache Superset for compelling dashboards.
Staying updated with emerging technologies, I'm proficient in AWS, Azure, and Google Cloud. In summary, as a dedicated Data Engineer, I bring extensive experience in the entire data engineering lifecycle. Let's collaborate to transform your data into a strategic asset for success!
